# 贪吃蛇游戏 🐍

经典怀旧贪吃蛇游戏，网页版即开即玩。

## 功能特点

- 🎮 经典贪吃蛇玩法
- 🏆 最高分本地记录
- ⚡ 随得分增加速度加快
- 📱 支持移动端触摸控制
- 🎨 炫酷霓虹视觉效果
- ⏸️ 暂停/继续功能

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始游戏"按钮
3. 使用方向键或 WASD 控制蛇的移动方向
4. 吃到红色食物得分，撞墙或撞到自己游戏结束

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上 |
| ↓ / S | 向下 |
| ← / A | 向左 |
| → / D | 向右 |
| 空格 | 暂停/继续 |

## 游戏规则

- 吃到食物得 10 分
- 蛇身变长
- 游戏速度逐渐加快
- 撞到墙壁或自己游戏结束

## 技术栈

- 纯 HTML + CSS + JavaScript
- Canvas 绘图
- LocalStorage 保存最高分
- 无需任何依赖或构建工具
